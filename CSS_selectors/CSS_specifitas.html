Hogyan működik a CSS specificitás?

A specificitást egy súlypontrendszer segítségével számítjuk ki, amely három komponensből áll:

  • ID-k száma - legmagasabb súly
  • Osztályok, attribútumok és pszeudo-osztályok száma - közepes súly
  • Elemek és pszeudo-elemek száma - legalacsonyabb súly

Az összehasonlítás balról jobbra történik: először az ID-k számát hasonlítjuk össze, majd ha az egyenlő, az osztályokét, végül az elemekét.
